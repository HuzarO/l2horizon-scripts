# Contributing to L2 Horizon Scripts

Thank you for considering contributing to this project!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/l2horizon-scripts.git`
3. Install dependencies: `npm install && cd editor && npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Code Style

- Use ES6+ syntax
- Follow existing code formatting
- Add comments for complex logic
- Use descriptive variable names

## Testing

- Test with actual L2 data files
- Verify all editor features work
- Check that scripts run without errors
- Test both inline and dialog editing

## Pull Request Process

1. Update README.md if needed
2. Test all changes thoroughly
3. Commit with clear messages
4. Push to your fork
5. Create a pull request with description

## Reporting Issues

When reporting bugs, include:
- Steps to reproduce
- Expected vs actual behavior
- Error messages/screenshots
- Your environment (OS, Node version)

## Questions?

Open an issue for discussion before major changes.
